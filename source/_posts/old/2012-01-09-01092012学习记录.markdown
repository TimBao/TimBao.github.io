  今天又学到一招STL的用法，嘿嘿。关于遍历容器的时候删除迭代器的方法：   例如有一个vector<int>; vec，及指向该vector的迭代器it；要删除其中值等于5的所有项，
  
  
 
 for(it = vec.begin(); it != vec.end(); it++)
{
    if(5 ==*it )
    {
        vec.erase(it++);//important
    }
}
 
 
 
  比我以前的做法效率要高很多。利用的++运算符的特性，以前怎么就没想到呢？看来看高手的源码还是很有用滴！
